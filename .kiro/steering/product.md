# Product Overview

Wandr is a travel itinerary planning application that helps users create personalized trip plans based on their destination, preferences, and weather conditions.

## Core Features

- Trip creation with destination selection and activity preferences
- Weather-aware recommendations (filters outdoor activities during rain)
- Automated itinerary organization across multiple days with time slots (Morning, Afternoon, Evening)
- Budget estimation using linear regression
- Interactive map visualization of planned locations
- Trip gallery for managing multiple saved trips
- Daily itinerary view with activity details

## User Flow

1. Users create trips by selecting a city, duration, and activity preferences
2. System fetches real-time weather and filters activities accordingly
3. Activities are automatically organized into daily time slots
4. Users can view trips on an interactive map, browse daily schedules, and check weather
5. All trips are saved in session state for easy access and management
